# Laboratorio 2

## Punto 2.1
Para el punto 2.1, se me ocurrió hacer una clase Persona para un sistema de registro, ya sea del estilo Renaper (Argentina) o similar. Creé la clase Persona con sus campos y propiedades accesibles mediante getters y setters, su constructor al cual se le pasan los parámetros de nombre y edad, y estableciendo el ID con un randomizador (método privado). También creé su destructor, el cual lanza un mensaje en consola.

## Punto 2.2
Procedo a crear los métodos para manejar strings: StringToCharArray y CharArrayToString para convertir un string en un array de caracteres y viceversa; GetSubstring para obtener un substring a partir de un string dado, la posición inicial del nuevo string y su longitud; CompareStrings para comparar strings y verificar si son iguales; y SearchString para buscar un string dentro de otro.
